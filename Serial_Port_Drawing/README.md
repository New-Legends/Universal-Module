本模组float协议开始，以类继承为核心，开放cpp和c两个版本的接口函数
尽量少的应用非官方头文件，即头文件从简
项目文件夹名：Serial_Port_Drawing


C语言接口封装逻辑链：
大前提：C++类实例化的对象的属性在内存中是一块连续地址
	    （是“C++的类是基于struct扩展实现的”的细化描述）

小前提：调用对象可以使用指针进行调用
	    （是“struct是一种数据类型”的扩展使用）

结论   ：可以使用指针对C++的对象进行封装交给C使用


C调用C++接口包装：
	.cpp					实现文件				实现基本功能
	.hpp					实现头文件			头文件整合
	wrapper.cpp			接口实现文件			指针调用类和extern"C"
	wrapper.h				接口头文件			头文件整合


C++的C适配工作流程：
	C++的功能实现
	C++对象创建、指针调用实现
	C语言头文件的整合

使用方法：
  C++版本：
    1、每一次调用实例化一个新的类
    2、调用具体操作函数
    3、释放调用的堆内存
  C语言版本：
    同上
